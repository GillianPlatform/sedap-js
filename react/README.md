# SEDAP React
A React library for creating UIs to interact with SEDAP.

## Acknowledgements
- Initial repo setup was assisted by [Ashish Kachhadiya's Medium article](https://medium.com/simform-engineering/building-a-component-library-with-react-typescript-and-storybook-a-comprehensive-guide-ba189accdaf5)
- Node arrangement and rendering is facilitated by [React Flow](https://reactflow.dev/)
- Node placement calculations are performed by [elkjs](https://github.com/kieler/elkjs)
